How to create a project

After login in to the platform you will se a button at the top right of the page with the text "Create porject" Click that button you will see how a form appears at the center of the page you need to fill out the form but first some requirements

1. Have a Database hosted on internet

2. This Database needs to be POSTGRESQL

How to create Endpoints

Once you have created a project and entered you will se the title of the project an empty list of Endpoint and the Database Configuration for creating a Endpoint click the button "Create Endpoint" when you click a form will appear at the center of the screen

This are important things to know when you are filling the new Endpoint Form

1. URL is the name of the endpoint for example getProducts, insertArticles etc...

2. in the SQL Query if you want to add parameters to it you need to put a '?' where you want to pass the argument always with space between the '?' and other characters
for example:
SELECT * FROM table WHERE id = ?
INSERT INTO table VALUES ( ? , ? )
DELETE FROM table WHERE id = ?

3. The params need to be writed in a special way this is an example "param1:int,param2:string" you need to put the param name and declare the type string or int the param1 needs to be the column name of the table, if you want more than 1 you will need a ',' to separate each param but wihout any spaces
